## 🎯 8. CEFR 적용 전략

### 8.1 현재 문제점

```
1. PRE-A1 상세도 부족
   - CEFR은 PRE-A1을 명확히 정의하지 않음
   - "Breakthrough" 또는 "Beginner" 범주
   - 500-700 어휘, 기초 구조 추정

2. 난이도 검증 부재
   - A1/A2 간 명확한 구분 기준 없음
   - 각 레벨의 "Can Do" 기술 미작성

3. 실제 학습자 수준 불명확
   - 합격 기준이 무의미해 보임
   - 진정한 레벨 달성 여부 확인 불가
```

### 8.2 개선 전략

#### 8.2.1 각 레벨별 "Can Do" 명시

```
PRE-A1 (Breakthrough):
✓ 기초적인 인사 나누기 (Hello, Goodbye)
✓ 간단한 자기 소개 (이름, 국가만)
✓ 기초 수사 인식 (1-10)
✓ 일반적인 물건 명명

A1 (Elementary):
✓ 일상 상황 단순 대화

✓ 여행/숙박 상황 대응

✓ 친숙하지 않은 상황 대응
✓ 추상 개념 이해
✓ 논증 이해 및 표현
✓ 정형식 글쓰기

B2 (Upper Intermediate):
✓ 전문 분야 토론
✓ 미묘한 의미 이해
✓ 설득적 글쓰기
✓ 고급 발표 능력
```
